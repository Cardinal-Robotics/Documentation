## Methods
1. Git command
2. VSCode UI

## Git command
1. Open console (ctrl + \`)
2. Run `git init`


    ![Git console](./images/init/initConsole.png)

## VSCode UI
1. Open `Source Control` on the left.


    ![Source control menu](./images/init/sourceControl.png)


2. Click `Initialize Repository`.
